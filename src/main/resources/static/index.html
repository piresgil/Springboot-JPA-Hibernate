Sobre

I am a dedicated Java developer with a strong focus on building robust, scalable, and user-friendly desktop applications. My main expertise lies in Java and its modern ecosystem — including JavaFX for UI development, Spring Boot for backend architecture, and JUnit for automated testing.

I hold a European Level 3 certification in Computer Management from the Professional School of Raia, Idanha-a-Nova, Portugal. My formal education provided a solid foundation in business management and programming with languages such as Pascal, Clipper, Visual Basic, and MySQL — along with basic Java. During that time, I developed a strong interest in software development by creating a data management system using Visual Basic as part of a school project.

Although I initially pursued other fields due to the lack of early opportunities in tech (2007), my passion for software development never faded. Since 2018, I have fully reengaged in the world of programming, taking a self-taught and project-driven approach — with a clear focus on mastering the Java ecosystem.

I’m actively seeking opportunities that allow me to grow as a Java developer and collaborate on meaningful, technically challenging projects.
